dbprocessing.DBlogging.change_logfile(logname=None)[source]¶

Switch to a new log file

This implements switching default logging from one filename to another, usually used to indicate the mission being processed. The filename will always start with dbprocessing_ and include a date string for the day.

Parameters
lognamestr, default “log”

Name to include in full log filename.
