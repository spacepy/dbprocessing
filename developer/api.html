These pages document the Python API for dbprocessing.

dbprocessing

dbprocessing main module.

DBfile

Database representations of data files and operations.

DBlogging

Support for logging information from the dbprocessing chain.

dbprocessing

High-level implementation of the dbprocessing processing queue.

DBqueue

General deque implementation, used for processing queue.

DBstrings

String handling for dbprocessing.

DButils

Direct interfacing to dbprocessing databases.

Diskfile

Information regarding a file as stored on disk.

inspector

Support for inspectors, which determine product type for a given file.

module

Wrapper for environment modules script (http://modules.sourceforge.net).

reports

Support for making reports from dbprocessing logs.

runMe

Support for running codes to make data files, as defined in database.

tables

Table definitions for dbprocessing.

Utils

Various utilities of use throughout this code.

Version

Handling of version numbers for codes and files, including comparisons.
